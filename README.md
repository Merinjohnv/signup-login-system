# Login & Signup System

A modern Login & Signup System built using HTML, CSS, and JavaScript with clean UI design and client-side authentication logic.

This project allows users to create an account, securely log in using stored credentials, and demonstrates frontend authentication concepts using LocalStorage.

---

## Features

- User-friendly Login and Signup forms
- Toggle between Login & Signup screens
- Client-side form validation
- Email format and password length validation
- Show / Hide password functionality
- Stores user credentials using LocalStorage
- Prevents login before signup
- Success and error feedback messages
- Responsive design for mobile and desktop

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API

---

## Concepts Practiced

- DOM Manipulation
- Event Handling
- Form Validation
- Conditional Rendering
- LocalStorage
- Authentication Flow
- Responsive UI Design

---

## How It Works

1. User creates an account using Signup form
2. Credentials are stored in LocalStorage
3. User logs in using saved email & password
4. Validation checks user input before authentication
5. Success or error messages are displayed dynamically
